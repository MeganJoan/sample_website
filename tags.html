Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

A block element is something that starts a new line.

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph or text
table table include table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

An inline element is something that flows with the text around it.

Tag Name Purpose Example Result
em emphasized make emphasized text <em>actually a dragon</em> actually a dragon
strong strong make strong text <strong>Dragons</strong> Dragons
a anchor make a hyperlink <a href="http://michaelhartl.com">Michael Hartl</a> Michael Hartl
img image include an image <img src="images/jaegerjagonsleep.jpg" alt="A photo of Jaeger sleeping" width="50">
